• The Logit Lens shows what each intermediate layer would have predicted
• Dimension traces show real hidden-state values changing layer by layer
• Signed delta heatmaps show exactly which dimensions increase/decrease at each layer
• Cross-step comparison shows whether the model reuses the same circuits for different tokens

NO PCA, NO dimensionality reduction — all values are real hidden-state dimensions.

🔍 What to Look For

1. Generated Output (📝): What did the model actually predict? How confident was it at each step?

2. Logit Lens (🔍): THE key visualization. Watch the correct answer emerge layer by layer. Early layers predict garbage, middle layers start forming the answer, final layers lock it in. Koch's C4: "inner layers compute."

3. Dimension Traces (📈): Real hidden dimensions changing across layers. Can you see dimensions that "specialize" for certain tokens?

4. Signed Delta Heatmaps (🔵🔴): Exact signed changes per dimension per layer. Blue = decrease, Red = increase. Trace the computation step by step.

5. Expansion/Contraction (📐): How much work does each layer do? Koch's C4 predicts middle layers should have the tallest bars.

6. Cross-Step Comparison (🔄): Does the model reuse the same "circuits" for different tokens, or adapt per token?
