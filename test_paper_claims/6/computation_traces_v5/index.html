🔍 What to Look For (mapped to Koch's conjectures)

1. Logit Lens (🔍): THE key visualization. Watch the correct answer emerge layer by layer. Early layers predict garbage, middle layers start forming the answer, final layers lock it in.
→ Koch's Conjecture 4: "Inner layers compute, outer layers translate." If true, the answer should emerge in the MIDDLE layers, not just appear at the end.

2. Deformation Magnitude (📐): How much work does each layer do?
→ Koch's Conjecture 4 predicts middle layers should have the tallest bars.

3. Signed Delta Heatmaps (🔵🔴): Blue=decrease, Red=increase. Trace the exact computation step by step.
→ Koch's Conjecture 1: "The geometry of the map provides indirect access to the model's reasoning."

4. Token Cloud (🌐): Do related tokens get closer? Unrelated ones farther?
→ Koch's Section 5: "Positive ORC indicates that neighboring tokens converge (a gravitational source); negative ORC indicates divergence."

5. Last-Layer Reversal (🔄): Does the last layer undo the morphing?
→ Koch's Conjecture 4: "The last layer often morphs the space back toward the embedding-layer geometry."

6. Rank Test (🔬): Is this real space-morphing or just a global shift?
→ Koch's framework is only meaningful if deformations are genuinely multi-dimensional and token-specific. If rank ≈ 1, it's just a bias — not interesting.

7. Wave Propagation (🌊): The cumulative "wave" of deformation.
→ Koch's Section 2.3: "The cumulative effect is a superposition of deformations — a 'wave' of space-morphing that propagates from layer 0 to layer L."

8. Cross-Step Comparison (🔄): Does the model reuse the same circuits?
→ Koch's Conjecture 5: "The Jacobian field carries holographic information." If true, you should see structured, repeating patterns across generation steps.

9. Full Map (🗺️): The raw data — actual hidden-state values for all tokens at all layers. This is what the model's "brain" actually looks like.
