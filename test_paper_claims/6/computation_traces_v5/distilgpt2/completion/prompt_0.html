🔄 Last-Layer Reversal Test

Koch's Conjecture 4 [[11]]: "The last layer often morphs the space back toward the embedding-layer geometry, undoing much of the intermediate deformation so that the result can be read off by the language-modeling head."

For each generation step, we measure the cosine similarity between each layer's delta and the first layer's delta. If the last layer reverses, its cosine should be negative (pointing in the opposite direction). Green = same direction. Red = reversal.

🔬 Is This Real Space-Morphing or Just a Global Shift?

Key question: When the model changes representations at each layer, does it move each token differently (genuine space-morphing), or just shift all tokens the same way (a global bias)?

We use the full hidden-state matrix (all tokens × all dimensions) at each layer, compute the delta, and check its effective rank. Rank ≈ 1 = just a global shift (boring). Rank >> 1 = genuine per-token morphing (interesting).
Deformation Diversity = what fraction of the total change is NOT explained by the mean shift. High = each token deformed differently. Low = just a global shift.

Koch's framework is only meaningful if the deformations are genuinely multi-dimensional and token-specific.
