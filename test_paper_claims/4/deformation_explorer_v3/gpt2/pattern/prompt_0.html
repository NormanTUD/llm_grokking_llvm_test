📐 Geometric Invariants of the Space Morphing

Three key measurements of how the space is being deformed at each layer:
1. Direction Coherence: Do all tokens move in the same direction? (>0 = together, <0 = apart, ~0 = independently)
2. Expansion/Contraction: Does the space stretch (>1) or shrink (<1)? 1.0 = no change.
3. Total Deformation: How much does the space change overall?

Koch's Conjecture 1: Tokens should move together (coherence > 0) because the model deforms the space itself, not individual points. Click legend items to isolate individual traces.

🔬 Is This Real Space-Morphing or Just a Global Shift?

Key question: When the model changes token representations at each layer, does it move each token differently (genuine space-morphing), or does it just shift all tokens the same way (a global bias)?

Effective Rank: If the delta matrix has rank ≈ 1, all tokens moved in the same direction — that's just adding a bias vector, not "morphing space." If rank >> 1, different tokens were deformed differently, which IS genuine space-morphing.
Deformation Diversity: What fraction of the total change is NOT explained by the mean shift? High = each token deformed differently. Low = just a global shift.

Koch's framework is only interesting if the deformations are genuinely multi-dimensional and token-specific. Hover for exact values at each layer.
